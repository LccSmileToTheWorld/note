# 数据类型

* 八大基本数据类型
  * 整型：byte，short，int，long
  * 浮点型：float，double
  * 字符型：char
  * 布尔型：boolean
* 引用数据类型
  * 类
  * 接口
  * 数组

# 变量分类

* 成员变量
  * 实例变量：不以static修饰
  * 类变量：以static修饰
* 局部变量
  * 形参
  * 方法体中定义的变量
  * 代码块中定义的变量



区别：

* 局部变量没有默认初始值，除了形参，在使用前都需要显示赋值；成员变量有默认初始值，可以不显示赋值。
  * 整型：0
  * 浮点型：0.0
  * 字符型：‘\U0000’（空字符）
  * 布尔型：false
  * 引用数据类型：null
* 局部变量：存放在虚拟机栈中；实例变量：存放在方法区中；类变量：存放在方法区中。
* 成员变量可以用权限修饰符修饰；局部变量不可以。
