 # 前进后退
作用：跳转到某一版本。



命令：
* 基于索引值操作：
    * git reset --hard <局部索引值>
    * 配合git reflog使用

![](.\11.png)

* 使用^：
     * git reset --hard HEAD^
     * 配合git reflog使用
     * 只能后退，一个^表示后退一步，n个^表示后退n步
* 使用~：
    * git reset --hard HEAD~n
    * 配合git reflog使用
    * 只能后退，表示后退n步



reset 命令的三个参数对比：

* --soft 参数
    * 仅仅在本地库移动HEAD 指针
* --mixed 参数
    * 在本地库移动HEAD 指针
    * 重置暂存区
* --hard 参数
    * 在本地库移动HEAD 指针
    * 重置暂存区
    * 重置工作区
